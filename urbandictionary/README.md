# UrbanDictionary

Look up Urban Dictionary definitions from Discord.

## Commands

- `[p]urban <term>` - show the top Urban Dictionary definition for a term.
- `[p]urban random` - show a random Urban Dictionary entry.
- `[p]ud <term>` - alias for `[p]urban <term>`.

The cog does not store user data. Lookup terms are sent to Urban Dictionary when a command is used.
