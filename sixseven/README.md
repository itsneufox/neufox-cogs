# SixSeven

Replies with a random annoyed comeback when someone says variants like `six seven`, `six-seven`, `sixseven`, `6 7`, `6-7`, `6.7`, or `67`.
Supports English and Portuguese response pools, with channel and category language overrides.

## Commands

- `[p]sixseven show` - Show current settings.
- `[p]sixseven toggle` - Enable or disable automatic replies.
- `[p]sixseven cooldown <seconds>` - Set the per-server reply cooldown.
- `[p]sixseven response <text>` - Set one fixed custom response.
- `[p]sixseven responses [en|pt]` - List the built-in random responses for a language.
- `[p]sixseven reset` - Restore random built-in responses.
- `[p]sixseven language default <en|pt>` - Set the server default language.
- `[p]sixseven language channel <#channel> <en|pt>` - Set a channel language.
- `[p]sixseven language category <category> <en|pt>` - Set a category language.
- `[p]sixseven language clearchannel <#channel>` - Remove a channel override.
- `[p]sixseven language clearcategory <category>` - Remove a category override.
